# IceLayers

Fast layer navigation for Ucupaint.

## Features

- Next Layer (`R`)
- Previous Layer (`Alt + R`)
- Wrap-around layer cycling
- Works directly on the active Ucupaint layer stack

## Requirements

- Blender 5.0+
- Ucupaint

## Usage

While in Texture Paint mode:

| Hotkey | Action |
|---------|---------|
| R | Next Layer |
| Alt + R | Previous Layer |

## Philosophy

IceLayers is designed around:

- Hotkeys first
- Minimal UI
- Faster painting workflows
- Staying focused on the canvas

The goal is simple:

Spend less time clicking through interfaces and more time creating art.

This addon was built to solve problems encountered during real production work and follows the belief that software should adapt to the artist, not the other way around.
